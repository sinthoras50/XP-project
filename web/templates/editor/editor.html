How to use

  • Use cursor to create rectangles of the areas to be extracted by the OCR application. In addition arrow and numpad arrow keys can be used to move/resize the current rectangle. Pressing 'P' creates a copy of the last added rectangle, which can then be manipulated with the controls.
  • Name the item that describes the rectangle. Filtered words can be filled with words that are not desired to be extracted.
  • Press the "Add" button when satisfied with the rectangle. Rectangles can be removed in the order in which they were added by pressing the "Undo" button.
  • Pressing the "Save" button will generate a json file containing all of the rectangle locations that can be used by the OCR application.
